# 学习笔记

## 1. 数组（Array）

### 1.1 定义

Java,c++: int a[100];

Python: list=[]  # 可以直接定义一个数组，也称列表

```python
list_1 = [1,2,3,4]
print(list_1)
```

### 1.2 时间复杂度

- 连续的内存空间，支持随机访问，时间复杂度$O(1)$
- insert、delete操作低效，平均时间复杂度$O(n)$

|        操作        | 时间复杂度 |
| :----------------: | :--------: |
| prepend(最前添加)  |  $$O(n)$$  |
|  append(最后添加)  |  $$O(1)$$  |
| lookup(查询，搜索) |  $$O(1)$$  |
|    insert(插入)    |  $$O(n)$$  |
|    Delete(删除)    |  $$O(n)$$  |

### 1.3 优缺点

优点：

- 地址连续，访问任意一项都是常数级别$O(1)$时间复杂度

缺点：

- 删除，添加元素时间复杂度高，一般为$O(n)$
- insert前需要保证数组的size足够，否则需要扩容，拷贝等低效操作
